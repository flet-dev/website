`ChartCrossPoint` class has the following properties:

### `color`

The fill [color](/docs/reference/colors) to use for the cross-mark(X).

### `size`

The size of the cross-mark.

Defaults to `8.0`.

### `width`

The thickness of the cross-mark.

Defaults to `2.0`.
